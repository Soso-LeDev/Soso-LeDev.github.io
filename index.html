Police — Metropolitan Police Service (MPS)

Standards professionnels

Honnêteté, intégrité, respect et responsabilité. Respectez le roleplay et les règles du serveur.

Réglementation du kit

  • Taser : Utilisation justifiée sur suspects menaçants.
  • Bâton : Défense contre suspects non coopératifs.
  • Radio : Pour communications. Utilisez les codes d'état.
  • PAVA : Spray incapacitant pour défense.
  • Menottes : Pour retenue.
  • Armes à feu : Seulement en cas de menace imminente pour la vie.

Codes d'état

  • State 0: Urgence — assistance immédiate requise
  • State 1: En service
  • State 2: En patrouille
  • State 3: Au poste — disponible
  • State 5: En route
  • State 6: Arrivé
  • State 9: Transport de prisonnier
  • State 11: Hors service
  • State 15: Reçu
  • State 16: Répéter le dernier

Divisions & Rangs

Divisions primaires : Response Policing, Armed Policing, Close Protection, Roads & Transport. Certaines divisions peuvent requérir des qualifications/permissions.

  • Trainee Constable (0 points): Handcuffs, Baton, Warrant card
  • Special Constable (2 points): Taser
  • Constable (5 points): Speed gun
  • Senior Constable (15 points): PAVA
  • Sergeant (50 points): Medical bag
  • Inspector (100 points): Clamp

Garde Royale (RG)

Guardsman : M1911 & L85A2 (munitions incluses)

Corporal : Permission spéciale d'ouvrir les portes du palais

La Garde Royale assure la protection du palais et le maintien de l'ordre dans les secteurs royaux.

Remarques & améliorations suggérées

  • Ajouter un menu hamburger pour mobiles si vous attendez beaucoup d'utilisateurs mobiles.
  • Proposer un JSON de configuration (liens, ranks, permissions) pour charger dynamiquement le contenu côté client.
  • Améliorer l'accessibilité des tableaux (filtres, recherche) pour trouver rapidement une commande.
  • Utiliser un CDN d'images ou stocker les assets sur GitHub/GDrive avec des URLs stables.
