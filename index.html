Police - Metropolitan Police Service (MPS)

Standards Professionnels

En tant que constable, adhérez à l'honnêteté, intégrité, respect et responsabilité.

Réglementation du Kit

Taser : Utiliser pour incapaciter suspects fuyant ou menaçants.

Baton : Pour la défense contre suspects non coopératifs.

Radio : Pour communications essentielles, codes d'état.

PAVA : Spray incapacitant pour défense.

Menottes : Pour retenir suspects résistants.

Armes à feu : Seulement pour menace imminente à la vie.

Codes d'État

  • Status 0: Urgence ; assistance immédiate requise
  • State 1: En service
  • State 2: En patrouille
  • State 3: Au poste ; disponible
  • State 5: En route
  • State 6: Arrivé
  • State 9: Transport de prisonnier
  • State 11: Hors service
  • State 15: Reçu
  • State 16: Répéter le dernier

Divisions

Divisions Primaires : Response Policing (quiz), Armed Policing (gamepass), Close Protection (gamepass), Uniformed Protection (via Close Protection), Escort Bikers (via Close Protection), Police Snipers (gamepass).

Divisions Spécialistes : MO7 (Taskforce), MO8 (Roads & Transport Policing Command), MO19 (Specialist Firearms Command). Requiert qualification policière et application.

Rangs et Outils

  • Trainee Constable (0 points): Handcuffs, Baton, Warrant card
  • Special Constable (2 points): Taser
  • Constable (5 points): Speed gun
  • Senior Constable (15 points): PAVA
  • Sergeant (50 points): Medical bag
  • Inspector (100 points): Clamp
  • Chief Inspector (150 points): British Transport Police
  • Superintendent (250 points): None
  • Chief Superintendent (350 points): None

Ambulance - London Ambulance Service (LAS)

LAS est une équipe payante (gamepass de 350 Robux) permettant de soigner les joueurs blessés et de sauver les downed players. C'est l'une des équipes les plus complexes du jeu.

Le manuel fournit des guidelines sur les rôles, structure de rangs, entraînement et usage d'équipement. Requiert un entraînement spécifique via le lien fourni.
